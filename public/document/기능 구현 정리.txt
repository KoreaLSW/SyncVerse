로그인
 - 구글로그인
   - 로그인 시 캐릭터 고르기
   - 닉네임 설정
 - 게스트로그인
   - 로그인 시 캐릭터 고르기
   - 랜덤 닉네임

메인
 - 캐릭터
   - 캐릭터 위치 실시간 표시
   - 차단 및 친구인 캐릭터는 닉네임 옆에 아이콘으로 표시
   - 게스트인 사용자도 별도로 표시
 - 전체 채팅기능
   - 내가 차단한 사용자는 메세지 안보임, 상대방도 나의 메세지가 안보임
 - 1:1 채팅기능
   - 상대방이 1:1채팅을 요청 시 수락해야 1:1 가능
 - 친구추가 및 삭제(게스트사용자는 구글 로그인 후 사용 가능)
   - 친구추가 시 상대방에게 알림
   - 친구삭제 시 상대방에게 모르게
 - 차단(게스트사용자는 구글 로그인 후 사용 가능)
   - 상대바엥게 알람 없음
   - 차단된 사용자들끼리 1:1 채팅 금지
 - 마이크
   - 마이크 on/off기능 추가
   - 차단한 사용자의 마이크는 들리지않음
   - 마이크로 이야기중일때 닉네임테두리에 초록색으로 표시
   - 특정사용자 마이크 끄기 기능
   - 캐릭터가 가까이에 있을때만 들리게

실시간 화이트보드
 - 채널 5개정도
 - 채널 인원수 제한기능
   - 정해진 인원수가 넘어가면 입장 불과
 - 실시간 그림표시
   - 사용자들이 그리는 그림들이 실시간으로 표시
 - 전체 채팅기능
   - 친구, 차단, 게스트 닉네임 옆에 아이콘으로 표시
 - 사용자별 마우스 위치
   - 마우스위치에 닉네임 표시
   - 친구, 차단, 게스트 닉네임 옆에 아이콘으로 표시
 
채팅방
 - 자유채팅방
   - 친구, 차단, 게스트 닉네임 옆에 아이콘으로 표시
   - 유저 입장 시 닉네임으로 입장했다는 표시
 - 1:N 채팅방생성(참가가능한인원수 지정(게스트사용자는 구글 로그인 후 사용 가능)
   - 채팅방 이름, 입장 시 비밀번호설정(필요시)
   - 입장 시 알림
   - 메인에서 1:1 채팅한 채팅방 리스트
 
 
게시판
 - 글쓰기(게스트사용자는 구글 로그인 후 사용 가능)
   - 제목, 글쓰기, 사진(최대 5장)
   - 글쓴사람만 삭제 가능
 - 댓글(게스트사용자는 구글 로그인 후 사용 가능)
   - 게시글 댓글 및 대댓글
   - 작성자만 삭제 가능

실시간화이트보드

채팅방
