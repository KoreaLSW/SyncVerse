로그인
 - 구글로그인
   - 로그인 시 캐릭터 고르기
   - 닉네임 설정
 - 게스트로그인
   - 로그인 시 캐릭터 고르기
   - 랜덤 닉네임

메인
 - 캐릭터(완료)
   - 캐릭터 위치 실시간 표시
   - 차단 및 친구인 캐릭터는 닉네임 옆에 아이콘으로 표시
   - 게스트인 사용자도 별도로 표시
 - 전체 채팅기능(완료)
   - 내가 차단한 사용자는 메세지 안보임, 상대방도 나의 메세지가 안보임
 - 1:1 채팅기능
   - 상대방이 1:1채팅을 요청 시 수락해야 1:1 가능
 - 친구추가 및 삭제(게스트사용자는 구글 로그인 후 사용 가능)(완료)
   - 친구추가 시 상대방에게 알림
   - 친구삭제 시 상대방에게 모르게
 - 차단(게스트사용자는 구글 로그인 후 사용 가능)(완료)
   - 상대방에게 알람 없음
   - 차단된 사용자들끼리 1:1 채팅 금지
 - 마이크
   - 마이크 on/off기능 추가
   - 차단한 사용자의 마이크는 들리지않음
   - 마이크로 이야기중일때 닉네임테두리에 초록색으로 표시
   - 특정사용자 마이크 끄기 기능
   - 캐릭터가 가까이에 있을때만 들리게

실시간 화이트보드
 - 채널 5개정도(완료)
 - 채널 인원수 제한기능(완료)
   - 정해진 인원수가 넘어가면 입장 불과
 - 실시간 그림표시(완료)
   - 사용자들이 그리는 그림들이 실시간으로 표시
   - 그림이 유지될수 있도록 현재상태 그림 저장 기능
 - 전체 채팅기능(완료)
   - 친구, 차단, 게스트 닉네임 옆에 아이콘으로 표시
 - 사용자별 마우스 위치(완료)
   - 마우스위치에 닉네임 표시
   - 친구, 차단, 게스트 닉네임 옆에 아이콘으로 표시
 
채팅방
 - 자유채팅방
   - 친구, 차단, 게스트 닉네임 옆에 아이콘으로 표시
   - 유저 입장 시 닉네임으로 입장했다는 표시
 - 1:N 채팅방생성(참가가능한인원수 지정(게스트사용자는 구글 로그인 후 사용 가능))(완료)
   - 채팅방 이름, 입장 시 비밀번호설정(필요시)(완료)
   - 입장 시 알림
 - 1:1 채팅방생성 및 리스트 (완료)
   - 메인에서 1:1 채팅방 포함 전체 채팅방 리스트(완료)
   - 유저 검색 후 1:1채팅 요청(완료)
   - 친구리스트에서 1:1채팅 요청(완료)
   - 같은 두 유저는 DM 방 1개만 유지(재요청 시 기존 방 열기)(완료)
   - 요청중 / 수락 / 거절 / 취소 / 만료 상태와 전이 규칙 명시(완료)
   - 
 - 공통사항
   - 메세지 삭제 기능(자신이 작성한 메시지만, 방장은 다른사람의 메시지도 삭제 가능(추후 기능 구현 예정))
   - 읽음처리(완료)
   - 사진전송기능(완료)
   - 알림기능(완료)
   - 최신순 저장 + 화면은 과거→현재, 무한스크롤 기준(offset/cursor) 명시(완료)
   

게시판
 - 글쓰기(게스트사용자는 구글 로그인 후 사용 가능)
   - 제목, 글쓰기, 사진(최대 5장)
   - 글쓴사람만 삭제 가능
 - 댓글(게스트사용자는 구글 로그인 후 사용 가능)
   - 게시글 댓글 및 대댓글
   - 작성자만 삭제 가능
   
설정
  - 프로필사진 설정
    - 사진 추가 및 수정 및 기본프로필으로 설정
    - 업로드 원본 권장: 512x512 (최소 256x256)
    - 크기는 96x96 / 용량은 최대 500KB
    - 미리보기 기능
  - 닉네임(nickname)수정 기능
    - 현재 닉네임 표시
    - 공백 금지 / 중복 금지 / 최대 12자 미만 / 특수문자 금지
    - 닉네임 변경 이력 남기기
  - 캐릭터 수정 기능
  - 알람
    - 알림 유형별 on/off(친구수락, 친구해제, 그룹초대 등 세분화)
  - 친구관리
    - 친구리스트
      - 해당 유저의 프로필 사진
      - 해당 유저의 닉네임
      - 해당 유저삭제버튼
      - 유저 검색
    - 해당 유저요청리스트
      - 해당 유저의 프로필 사진
      - 해당 유저의 닉네임
      - 유저 검색
    - 차단리스트
      - 해당 유저의 프로필 사진
      - 해당 유저의 닉네임
      - 차단 해제버튼
      - 유저 검색
  - 계정 및 보안
    - 프로필 공개 범외 설정
      - 전체/친구/차단
    - 온라인 상태 표시 on/off
    - 로그아웃
      - 세션에서 삭제
    - 회원탈퇴
      - 유저테이블에서 해당 유저 삭제
