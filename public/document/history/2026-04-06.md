## 삭제 UX 개선 (소프트 삭제 + 아이콘 버튼)

- 하드 삭제 대신 소프트 삭제로 변경
    - 삭제 시 메시지 본문을 `삭제된 메세지입니다`로 업데이트
    - 첨부 파일은 Cloudinary 및 `message_attachments`에서 정리
- 메시지 UI 반영
    - 삭제된 메시지는 이탤릭/흐린 톤으로 표시
    - 삭제된 메시지에서는 첨부/삭제 버튼 숨김
- 삭제 버튼 스타일 변경
    - 빨간색 쓰레기통 아이콘 버튼으로 교체
